- [Node](https://nodejs.org) > 10.3

## ‚öôÔ∏è Usage

Install npm dependencies

```sh
 npm install 
```

Run the kickstart command

```sh
npm run kickstart
```

**After the project has been kickstarted**

To start the development server

```sh
npm start
```

To build for production

```sh
npm run build
```

To preview the production build

```sh
npm run preview
```
